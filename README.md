# ICS Platform

This is ichebo-platform, Version 3 build, and link to the ecosystem architecture document.

## Stack

- Backend: Django 4.2 + Django REST Framework + PostgreSQL
- Frontend: Vanilla JS (mobile-first, IIFE modules)
- Server: Nginx + Gunicorn on VPS

See `/ichebo-platform/README.md` for local development setup.
