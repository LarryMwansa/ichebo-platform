# Apostolic Command Shell: Operational Manual (v2.4)

This document serves as the definitive guide for the **Apostolic Command Shell**, the sovereign high-density workspace for the Ichebo Institutional Calendar and Governance ecosystem.

---

## 1. Design Philosophy: "Apostolic Editorial"
The shell is built on the **Apostolic Editorial** design system, which prioritizes clarity, authority, and high-density information management.

*   **Sovereign Palette (Ink & Stone)**:
    *   **Ink (Dark Mode)**: Deep graphite (#0E0E0E) backgrounds with high-contrast typography, evoking the weight of institutional authority.
    *   **Stone (Light Mode)**: Parchment-toned surfaces (#F5F3F0) designed for long-form reading and clarity.
*   **The "Rule of Left"**: A signature 3px Red accent (`--accent-red`) applied to the left border of active navigational elements, signaling the current locus of command.
*   **Typography**:
    *   **Headers**: *Playfair Display* (Serif) for mandates and institutional titles.
    *   **Body**: *Inter* (Sans-serif) for functional data and tactical coordination.

---

## 2. Four-Column Grid Architecture
The workspace is organized into four distinct tactical columns, ensuring that the Steward never loses context while drilling into specifics.

1.  **Column 1: Primary Sidebar (Global Nav)**
    *   Ultra-slim switcher for primary applications (Dashboard, Desk, Records, Community, Settings).
    *   Contains the persistent "Rule of Left" indicator.
2.  **Column 2: Context Bar (App Explorer)**
    *   Dynamic column that changes based on the active app (e.g., Folder trees in Records, Kanban lanes in Community).
    *   **Command**: Toggle with `CMD/CTRL + [`.
3.  **Column 3: The Stage (Sovereign Canvas)**
    *   The primary workspace where mandates are drafted, data is analyzed, and the Knowledge Graph is visualized.
    *   Features a subtle watermarked institutional branding in the background.
4.  **Column 4: Options Bar (Sidecar & Metadata)**
    *   Integrated panel for metadata, Bible sidecar, and version history.
    *   **Command**: Toggle with `CMD/CTRL + ]`.

---

## 3. High-Performance Modes

### Focus Mode
Designed for deep work and drafting. Instantly collapses Columns 2 and 4 to maximize the Stage's horizontal real estate.
*   **Trigger**: `CMD/CTRL + \` or via Command Palette (`> Focus Mode`).

### Sovereign Shift (Dark/Light)
Real-time theme switching that updates all CSS design tokens across the platform.
*   **Trigger**: `CMD/CTRL + T` (shortcut) or `> Toggle Theme` in the palette.

---

## 4. The Command Palette (Search Overlay)
Accessible via `CMD/CTRL + /`, the palette is the central nervous system for institutional navigation.

*   **Registry Search**: Real-time HTMX-powered search across Mandates, People, and Journal entries.
*   **Command Mode (`>`)**: Type `>` to switch to system execution. 
    *   Example: `> Open Desk` navigates instantly to the Editorial Desk.
*   **Keyboard Navigation**: Full arrow-key (`↑/↓`) support with "Enter to Execute" logic.

---

## 5. Integrated Core Modules

### A. Ministry Dashboard
A strategic "Command Center" featuring high-density stat grids, real-time mission-critical alerts, and "Divine Intel" summaries.

### B. The Editorial Desk
The sovereign drafting environment for Mandates and Principles. Optimized for high-density editing with integrated metadata in the Options Bar.

### C. Apostolic Web (Knowledge Graph)
A D3.js-powered relationship visualizer that maps the connections between different mandates, entities, and divine patterns.

### D. Institutional Calendar
A centralized timeline of community activities and mandates, accessible via a dedicated sidebar or the global Stage.

### E. Bible Sidecar
A persistent sidecar in Column 4 that allows Stewards to reference scripture directly alongside their drafting or record-keeping.

## 6. Essential Shortcuts Reference

| Shortcut | Action |
| :--- | :--- |
| `CMD + /` | Open Command Palette |
| `CMD + [` | Toggle Context Sidebar (Col 2) |
| `CMD + ]` | Toggle Options Sidecar (Col 4) |
| `CMD + \` | **Focus Mode** (Collapse Sidebars) |
| `ESC` | Close Palette / Dismiss Modals |
| `↑ / ↓` | Navigate Palette Results |
| `Enter` | Execute Selected Command |
