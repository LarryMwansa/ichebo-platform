## The Five Strategic Shifts

### 1. **Induction is Level Zero** (12-week formation stage, not optional signup)
- Users enter Induction Tenant on signup
- Week 1-4: Orientation to governance + system
- Week 5-8: Gifting assessment + foundations
- Week 9-12: Placement assessment + transition prep
- Auto-advance to Level 1 on completion

### 2. **Learning Programmes Drive Advancement** (measurable, repeatable pathways)
- Certificate (Level 1, 1 year)
- Diploma (Level 2, 3 years)
- Degree (Level 3, 4 years)
- Masters (Level 4, 4-5 years)
- Doctorate (Level 5, 7 years)
- Signal handlers auto-detect completion → auto-promote

### 3. **Multi-Tenancy from Day One** (governance architecture, not data feature)
- Induction Tenant = system-wide entry point
- Permanent Tenant = where you belong (geographic or functional community)
- Level 0 sees Induction content only
- Level 1+ see their assigned tenant(s) + collaborating networks
- Level 4+ see Handbook across all tenants

### 4. **Stewards Become Community Coordinators** (pastoral responsibility, not abstract governance)
- Level 3 stewards get Community Coordinator dashboard
- See member roster, formation progress, recent activities
- Can assign roles (Shepherd, Net Caster, Net Mender)
- Tied to specific Sceptre Community (tenant)

### 5. **App Drawer Shows Clear Unlock Paths** (transparent progression)
- Locked apps show modal: "Requires Level 2. Enrol in Certificate (10 courses, 8 weeks) to advance."
- Every app is a visible milestone
- Users control their own progression
